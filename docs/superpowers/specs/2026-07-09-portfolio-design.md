## Техника
- **Стек:** vanilla HTML/CSS/JS, без сборки. CDN: Three.js (шейдер-хиро,
  3D-телефон), GSAP + ScrollTrigger (скролл-фильм), Lenis (плавный скролл).
  Реальные библиотеки, не canvas-наброски (жёсткое правило проекта).
- **Мобайл:** device detection → облегчённые сцены (меньше частиц, упрощённые
  шейдеры), обязательный `prefers-reduced-motion`, fallback без WebGL
  (статичная тёмная сцена, контент полностью доступен).
- **Ассеты:** скриншоты OASIS снимаются с прода (Playwright/Preview) — свои,
  лицензионно чистые. Атмосферные фото если нужны — Unsplash/Pexels.
  AI-генерация допустима для кастомных визуалов. Pinterest — только как
  референс настроения, никогда как источник файлов.
- **Хостинг:** GitHub Pages (статика). Бэкенд не нужен: CTA — ссылка на
  Telegram. Лимиты Pages (1 ГБ, ~100 ГБ/мес) заведомо не достигаются.
- **Производительность:** бюджет веса и Lighthouse-проверка перед сдачей.
